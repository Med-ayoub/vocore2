# Welcome

## Download 

```sh
git clone https://github.com/openwrt/openwrt.git
cd openwrt
git checkout v18.06.5
cd ./package
git clone https://github.com/Med-ayoub/vocore2.git
```

## configure

```sh
cd path/to/openwrt
./package/vocore2/openwrt.1806/vocore2-setup.sh

```

##compile 

```sh
make download
make 
```
